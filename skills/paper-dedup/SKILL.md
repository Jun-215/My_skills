# 论文降重专家

学术论文深度降重技能：重构文本结构与表达，保持学术严谨性，显著降低查重率。

## 工作流程

论文降重遵循以下步骤：

1. 读取源文件内容
2. 分析与理解核心内容
3. 结构重建与深度改写
4. 增加原创性表达
5. 学术规范检查
6. 输出到"论文去重"文件夹

## 步骤1：读取源文件

- 读取用户指定的论文文件（支持 `.md`、`.txt`、`.docx` 等格式）
- **绝对不能修改源文件**，所有输出写入独立的"论文去重"文件夹
- 若用户直接粘贴文本，则以该文本为输入

## 步骤2：分析与理解

对输入内容进行深度分析：

- 提炼核心观点、研究目的、方法论、数据和结论
- 识别关键术语和不可替换的专业表达
- 标记引用内容（引用部分需保持原意，不得曲解）
- 理解段落间逻辑关系和论证链条

**禁止**：逐句替换、简单同义词替换、仅调整语序。

## 步骤3：结构重建与深度改写

这是降重的核心步骤。对每个段落，从以下策略中至少使用3种。详细策略与示例见 [references/rewrite-strategies.md](references/rewrite-strategies.md)。

| 策略 | 说明 |
|------|------|
| 句式变换 | 主动⇄被动、拆句、合句、倒装 |
| 逻辑重排 | 改变"背景→问题→方法→结果"的顺序 |
| 因果解释 | 补充原因分析和推理过程 |
| 对比分析 | 加入与其他方法/结果的对比 |
| 举例说明 | 增加具体场景或实例辅助说明 |
| 抽象⇄具体 | 抽象概括化或具体展开化 |
| 列表/流程化 | 将叙述改为列表或流程描述 |
| 跨语言打散 | 模拟"中→日→中"句式结构变化效果 |

## 步骤4：增加原创性

适度加入以下内容以提高原创性：

- **方法动机**：解释为什么选择该方法
- **局限性说明**：补充研究局限性的讨论
- **实验观察**：添加对实验结果的分析性描述
- **逻辑推理**：加入个人分析或合理推理

**底线**：不编造数据、不虚构结论、不歪曲引用含义。

## 步骤5：学术规范检查

改写完成后执行以下检查：

- [ ] 学术语气一致，无口语化表达
- [ ] 专业术语准确无误
- [ ] 核心观点、数据、结论与原文一致
- [ ] 引用含义未被改变
- [ ] 未删除重要信息
- [ ] 逻辑连贯，上下文衔接自然
- [ ] 段落间过渡流畅

## 步骤6：输出

1. 在源文件同级目录下创建"论文去重"文件夹（若不存在）
2. 将降重后的论文保存至该文件夹，文件名格式：`原文件名_降重.扩展名`
3. 保持原文件不变

## 注意事项

- 每次改写需覆盖整个段落的逻辑和表达，而非局部词汇替换
- 对于高重复率的段落，优先使用"逻辑重排 + 因果解释 + 句式变换"组合
- 数学公式、图表标注、参考文献格式不做改动
- 若用户指定了特定段落，仅对该段落进行处理
